# 차트 기능 개선 안내

## ✅ 새로 추가된 기능

### 1. 기간 선택
차트 상단에 기간 선택 버튼이 추가되었습니다:
- **1주**: 최근 1주일 데이터
- **1개월**: 최근 1개월 데이터
- **3개월**: 최근 3개월 데이터
- **6개월**: 최근 6개월 데이터
- **1년**: 최근 1년 데이터
- **전체**: 전체 데이터 (기본값)

### 2. 차트 확대/축소
- **확대 버튼**: 차트 높이를 600px로 확대
- **축소 버튼**: 차트 높이를 400px로 축소
- 더 자세한 분석이 가능합니다

### 3. Brush 줌 기능
- 차트 하단에 스크롤바(Brush) 추가
- 드래그하여 특정 구간만 확대해서 볼 수 있습니다
- 원하는 기간을 선택하여 상세 분석 가능

### 4. 상세 통계 정보
차트 하단에 통계 정보가 표시됩니다:

**가격 차트:**
- 현재 가격
- 기간 최고가
- 기간 최저가
- 변동률 (시작 대비)

**지표 차트 (RSI/FGI):**
- 현재 RSI 값 및 상태 (과매수/과매도/중립)
- 평균 RSI
- 현재 Fear & Greed Index
- 평균 Fear & Greed Index

### 5. 개선된 툴팁
- 마우스를 차트 위에 올리면 상세 정보 표시
- 날짜, 가격, 지표 값 등 정확한 정보 제공
- 색상으로 상태 표시 (과매수/과매도)

### 6. 참조선 (Reference Lines)
**가격 차트:**
- 현재 가격 기준선 (초록색 점선)

**지표 차트:**
- RSI 70 (과매수 기준선 - 빨간색)
- RSI 30 (과매도 기준선 - 초록색)
- RSI 50 (중립 기준선 - 회색)

## 📊 차트 타입별 기능

### 가격 차트 (PriceChart)
- VIG/QLD 가격 라인
- 200일 이동평균선 (주황색 점선)
- 기간별 필터링
- 확대/축소
- Brush 줌

### 지표 차트 (IndicatorChart)
- RSI 지표 (파란색)
- Fear & Greed Index (주황색)
- 과매수/과매도 구간 표시
- 기간별 필터링
- 확대/축소
- Brush 줌

## 🎯 사용 방법

### 1. 기간 선택
```
1. 차트 상단의 기간 버튼 클릭
2. 원하는 기간 선택 (1주, 1개월, 3개월, 6개월, 1년, 전체)
3. 차트가 선택한 기간의 데이터로 필터링됨
```

### 2. 차트 확대
```
1. 차트 우측 상단의 "확대" 버튼 클릭
2. 차트 높이가 600px로 확대됨
3. "축소" 버튼으로 원래 크기로 복귀
```

### 3. Brush 줌 사용
```
1. 차트 하단의 Brush (스크롤바) 확인
2. 좌우 끝점을 드래그하여 원하는 구간 선택
3. 선택한 구간만 차트에 표시됨
4. Brush를 전체로 다시 드래그하여 전체 보기 복귀
```

### 4. 상세 정보 확인
```
1. 마우스를 차트 위에 올림
2. 툴팁에 해당 날짜의 상세 정보 표시
3. 정확한 가격, 지표 값 확인 가능
```

## 💡 활용 팁

### 단기 분석
- "1주" 또는 "1개월" 선택
- 최근 시장 동향 파악
- 단기 매매 타이밍 분석

### 중기 분석
- "3개월" 또는 "6개월" 선택
- 추세 분석
- 200일 이동평균선과 가격 비교

### 장기 분석
- "1년" 또는 "전체" 선택
- 장기 추세 파악
- 백테스트 결과와 비교

### 특정 구간 상세 분석
- Brush를 사용하여 특정 구간 선택
- 중요한 시점(급등/급락) 집중 분석
- 패턴 인식
