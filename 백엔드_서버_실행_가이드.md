# 백엔드 서버 실행 가이드

## 문제 해결 완료

### 발견된 문제
```
ModuleNotFoundError: No module named 'feedparser'
```

### 해결 방법

#### 1. 패키지 설치

```powershell
cd backend
.\venv\Scripts\Activate.ps1
pip install feedparser beautifulsoup4 vaderSentiment nltk
```

또는 requirements.txt 전체 설치:

```powershell
pip install -r requirements.txt
```

#### 2. 서버 실행

**방법 1: 배치 파일 사용**
```powershell
.\백엔드_실행.bat
```

**방법 2: 직접 실행**
```powershell
python main.py
```

**방법 3: uvicorn 직접 실행**
```powershell
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 필요한 패키지 (최소)

- `feedparser` - RSS 파싱 (필수)
- `beautifulsoup4` - HTML 파싱 (필수)
- `vaderSentiment` - 감성 분석 (필수)
- `nltk` - 텍스트 처리 (필수)
- `lxml` - XML 파서 (권장)

### 서버 확인

서버가 정상 실행되면:
- http://localhost:8000 - API 루트
- http://localhost:8000/docs - API 문서 (Swagger UI)
- http://localhost:8000/health - 헬스 체크
