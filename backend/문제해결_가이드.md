# 백엔드 서버 실행 문제 해결 가이드

## 문제: 백엔드 서버가 실행되지 않음

### 원인
`ModuleNotFoundError: No module named 'feedparser'` - 필수 패키지가 설치되지 않음

### 해결 방법

#### 1. 패키지 설치

```powershell
cd backend
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

또는 개별 설치:

```powershell
pip install feedparser beautifulsoup4 lxml vaderSentiment nltk newspaper3k requests-html selenium
```

#### 2. 설치 확인

```powershell
python -c "import feedparser, bs4, vaderSentiment, nltk; print('모든 패키지 설치 완료')"
```

#### 3. 백엔드 서버 실행

```powershell
python main.py
```

또는 배치 파일 사용:

```powershell
.\백엔드_실행.bat
```

### 필요한 패키지 목록

- `feedparser==6.0.10` - RSS 파싱
- `beautifulsoup4==4.12.2` - HTML 파싱
- `lxml==5.1.0` - XML/HTML 파서
- `vaderSentiment==3.3.2` - 감성 분석
- `nltk==3.8.1` - 텍스트 처리
- `newspaper3k==0.2.8` - 뉴스 처리
- `requests-html==0.10.0` - 웹 스크래핑 (선택)
- `selenium==4.15.2` - 브라우저 자동화 (선택)

### NLTK 데이터 다운로드

NLTK 데이터는 자동으로 다운로드되지만, 수동으로도 가능:

```python
import nltk
nltk.download('punkt')
nltk.download('stopwords')
```

### 주의사항

1. **가상환경 활성화**: 반드시 가상환경을 활성화한 후 설치
2. **Python 버전**: Python 3.8 이상 필요
3. **인터넷 연결**: 패키지 다운로드를 위해 인터넷 연결 필요

### 빠른 해결 방법

1. `백엔드_실행.bat` 파일 실행
2. 자동으로 패키지 확인 및 설치 시도
