# 실행 방법

## 🚀 빠른 시작

### 백엔드 실행 (필수)

**방법 1: 실행 스크립트 사용 (추천)**
```powershell
.\start_backend.ps1
```

**방법 2: 수동 실행**
```powershell
cd backend
.\venv\Scripts\Activate.ps1
python main.py
```

백엔드는 `http://localhost:8000`에서 실행됩니다.

---

### 프론트엔드 실행

**먼저 Node.js가 설치되어 있어야 합니다!**

Node.js 설치 여부 확인:
```powershell
node --version
```

**Node.js가 설치되어 있다면:**

**방법 1: 실행 스크립트 사용 (추천)**
```powershell
.\start_frontend.ps1
```
> 자동으로 npm install도 실행됩니다.

**방법 2: 수동 실행**
```powershell
cd frontend
npm install  # 첫 실행 시에만 필요
npm run dev
```

프론트엔드는 `http://localhost:3000`에서 실행됩니다.

---

## 📝 전체 실행 순서

### 1단계: 백엔드 실행
새 PowerShell 창에서:
```powershell
cd "C:\Users\WIN\Desktop\new project"
.\start_backend.ps1
```

백엔드가 정상적으로 실행되면 다음과 같은 메시지가 나타납니다:
```
INFO:     Uvicorn running on http://0.0.0.0:8000
```

### 2단계: 프론트엔드 실행
**새 PowerShell 창**에서 (백엔드 실행 중인 창과 별도):
```powershell
cd "C:\Users\WIN\Desktop\new project"
.\start_frontend.ps1
```

프론트엔드가 정상적으로 실행되면:
```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:3000/
```

### 3단계: 브라우저에서 접속
- 프론트엔드: http://localhost:3000
- 백엔드 API 문서: http://localhost:8000/docs

---

## ⚠️ 주의사항

1. **두 개의 터미널 필요**
   - 백엔드와 프론트엔드는 각각 별도의 PowerShell 창에서 실행해야 합니다.

2. **백엔드 먼저 실행**
   - 프론트엔드는 백엔드 API를 호출하므로, 백엔드를 먼저 실행하는 것을 권장합니다.

3. **Node.js 설치 확인**
   - 프론트엔드를 실행하려면 Node.js가 설치되어 있어야 합니다.
   - 설치되지 않았다면: https://nodejs.org/ 에서 LTS 버전 다운로드 및 설치

## 🔧 문제 해결

### 백엔드 실행 오류
```powershell
# 가상환경 재생성
cd backend
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
python main.py
```

### 프론트엔드 실행 오류
```powershell
# Node.js 확인
node --version

# npm 패키지 재설치
cd frontend
rm -r node_modules  # 또는 삭제
npm install
npm run dev
```

### 포트 충돌
- 백엔드(8000) 또는 프론트엔드(3000)가 이미 사용 중이면 다른 프로그램을 종료하세요.

---

## ✅ 정상 실행 확인

### 백엔드 확인
- 브라우저에서 http://localhost:8000 접속
- `{"message": "VIG-QLD 스위칭 투자 어드바이저 API", "version": "1.0.0"}` 메시지가 보이면 성공

### 프론트엔드 확인
- 브라우저에서 http://localhost:3000 접속
- 대시보드 페이지가 로드되면 성공
